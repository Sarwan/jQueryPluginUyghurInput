JQuery ئۇيغۇرچە كىرگۈزگۈچ قوشۇلمىسى

ئىشلىتىش ئۇسۇلى:

  • Jquery نىڭ 1.x نەشرىنى چۈشۈرۈپ تور بېتىڭىز بىلەن بىر مۇندەرىجىدە قويۇڭ.(2.x نەشرى ie6,7,8 نى قوللىمايدۇ.)

  • Jquery.Plugin.UyghurInput.js نى چۈشۈرۈپ تور بېتىڭىز بىلەن بىر مۇندەرىجىدە قويۇڭ.

  • بەت بېشىغا بۇ كودنى قىستۇرۇڭ:
    <header>
        <script src="jquery-1.10.2.js"></script>
        <script src="Jquery.Plugin.UyghurInput.js"></script>
        ...
    </header>
  • ئايرىم بىر script بۆلىكى قىستۇرۇپ ئىچىگە تۆۋەندىكى كودنى يېزىڭ:

        <script type="text/javascript">
            $(document).ready( function () {
                    //
                });
        </script>
  • بارلىق تېكىست رامكىلىرىغا ئۇيغۇرچە كىرگۈزۈش(قىزىل رەڭدە بويالغان كود):

        <script type="text/javascript">
            $(document).ready( function () {
                   $(document).Uyghurinput();
                });
        </script>
  • مەلۇم بىر رامكىغىلا كىرگۈزۈش:

        <script type="text/javascript">
            $(document).ready( function () {
                   $("#txtName").Uyghurinput();
                });
        </script>

  • خاسلىق نامىغا ئاساسەن كىرگۈزۈش:

        <script type="text/javascript">
            $(document).ready( function () {
                   $(".uyghur").Uyghurinput();
                });
        </script>
  • بالا كونتروللارنىڭ ھەممىگە كىرگۈزۈش:

        <script type="text/javascript">
            $(document).ready( function () {
                   $("#inputArea").Uyghurinput();
                });
        </script>
